# Roadmap 90 jours

## Jours 1-15 — MVP fonctionnel en ligne

- Auth, profils, annonces, messagerie.
- Upload photos et modération minimale.
- Déploiement Cloudflare + Supabase.

## Jours 16-30 — Transaction et confiance

- Offres / contre-offres.
- Paiement sandbox Stripe Connect.
- Avis et réputation.

## Jours 31-60 — Services entretien/réparation

- Catalogue services.
- Workflow devis → validation → exécution → preuve photo.
- Commissions automatiques.

## Jours 61-90 — Intelligence produit

- Score "bonne affaire".
- Alertes prix.
- Tableau de bord opérations (conversion, litiges, délai service).

## KPI de lancement

- Taux annonce -> offre.
- Taux offre -> transaction.
- Délai moyen traitement service.
- NPS acheteurs/vendeurs.

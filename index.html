Manip pour publier la bêta

  1. Installer Node.js 18+ sur le serveur.
  2. Copier le repo puis lancer node server.js.
  3. Mettre un reverse-proxy Nginx sur le port 3000.
  4. Activer HTTPS (Let's Encrypt) avant ouverture publique.
  5. Passer la base JSON vers PostgreSQL pour la prod.
